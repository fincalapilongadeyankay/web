FAQ: Preguntas Frecuentes

Windows 8 me avisa de que esta descarga es potencialmente peligrosa
Este es el comportamiento por defecto de Windows 8 para todas las descargas que no disponen de un certificado digital o no están aprovadas por Microsoft. Recientemente adquirimos un certificado digital de clase 2, que certifica nuestra autoría y que la descarga no fué manipulada. No obstante, mientras Microsoft no apruebe la firma, será necesario seguir autorizando el programa manualmente. El programa es perfectamente seguro para su uso.
También puedes descargar la versión Zip y descomprimirla en cualquier directorio. Generará una versión portable que podrás usar si ya tienes instalado Java 8 en tu ordenador.
¿Instalar el programa va a desconfigurar mi ordenador? ¿Podré luego desinstalarlo fácilmente?
El programa lleva incluido su propia versión de Java y no va a interferir con la copia de Java del sistema si la tienes o con ninguna otra configuración. Se puede desinstalar como cualquier otro programa. Más aún, tienes la opción de instalar la versión "portable" que está auto-contenida y la puedes mover a cualquier sitio (como una llave USB) para usar desde cualquier ordenador sin necesidad de instalar nada.
¿Podré crear mi propio servidor de Salas privado?
Si, fácilmente en tu propio ordenador, con tan solo especificar que el tipo de sala privada sea "Chat Directo Seguro". Estas salas están accesibles con una llave encryptada especial que puedes compartir con tus amigos. Ni siquiera es necesario que conectes a la red de Filetopia.
¿Puedo pegar imágenes en el chat y usar mis propios emoticonos animados?
Puedes copiar/pegar imágenes del chat y capturar iconos que han puesto otras personas. Para poder introducir tus propias imágenes e iconos necesitas ser habilitado por un operador. Esto es así para evitar problemas por contenido inapropiado en las comunidades.
¿Cuantas salas puedo tener?
Por defecto dispones de una sala pública y tantas salas privadas como quieras. En caso de tener una necesidad justificada para más salas públicas puedes ponerte en contacto con cualquier operador del sistema que te habilitará las salas que necesites. Puedes gestionar, no obstante, tantas salas de otros usuarios como tengas permiso (estatus de operador)
